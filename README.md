# Kennedy Mokaya | CSS Styling Showcase 🚀

An advanced, responsive portfolio interface built with **Vanilla HTML5, CSS3, and JavaScript**. This project serves as a hub to showcase my software development tools and academic work in Data Science at Meru University.

## 🛠️ Key CSS Techniques Demonstrated

* **Glassmorphism & Frosted Glass:** Utilizing `backdrop-filter: blur()` and semi-transparent RGBA layers to create depth.
* **Dynamic Interactions:** Custom `cubic-bezier` timing functions for high-end, smooth transitions.
* **Floating Label Logic:** Pure CSS form validation states (`:focus` and `:valid`) for an interactive contact experience.
* **Responsive Grid Architecture:** A mobile-first approach using **CSS Grid** and **Media Queries** to ensure 100% compatibility across devices.
* **Glass-Neon Aesthetic:** Strategic use of `box-shadow` and `radial-gradient` to create glowing background "blobs."
